# BTech Resources Repository

This repository contains structured academic resources for B.Tech students.

## 📂 Categories

* Notes
* Syllabus
* Previous Year Questions (PYQ)

## 🏫 Branches

* CSE
* ECE
* Mechanical
* IT

## 📌 Structure

Each branch contains 8 semesters, and each semester includes relevant study materials.

## 🚀 Usage

All files are served via CDN using jsDelivr for fast access in the website.

## 🤝 Contribution

Students can contribute notes, PYQs, and other useful resources.
